# Welcome to My Blog

This is an example blog post. You can write your content here using Markdown.

## Getting Started

To create a new blog post, simply add a new `.md` file to the `src/content/blog/` folder with the following frontmatter:

```yaml
---
title: "Your Post Title"
description: "A brief description of your post"
pubDate: 2026-02-03
tags: ["tag1", "tag2"]
draft: false
---
```

## Markdown Features

You can use all standard Markdown features:

- **Bold text**
- *Italic text*
- `Code snippets`
- [Links](https://example.com)
